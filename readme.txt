Alerts WordPress moderator of impending post for review.  Allows poster (e.g. Contributor role) to decide when a draft is ready with a simple checkbox.  *No options*

== Installation ==

1. Upload `notify-draft.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
